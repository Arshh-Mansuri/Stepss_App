Critical implementation rules

App namesOnly via Label(token:). Token chips in mockups are placeholders — never build custom icon rendering.
PricingRead from App Group UserDefaults. Defaults: 500 / 900 / 1,600 pts. Never hardcoded in view layer.
Conv. rateFixed at 1 step = 1 pt for MVP. Stored in pointsPerStep UserDefaults. No UI to change it in v1.
Shield actionsShieldActionExtension only opens steplock://spend. Zero unlock logic in any extension.
Balance readsAlways from GRDB PointsBalance row id=1. Never cached in SwiftUI view state.
Shield themeAlways dark — hardcoded. Ignores system light/dark mode. Self-contained colour set.
Touch targets44×44pt minimum (Apple HIG). Extend hit area beyond visual bounds when icon is smaller.
Spend confirmButton disabled until app + duration selected AND balance sufficient. Disabled immediately on first tap (prevents double-spend).
